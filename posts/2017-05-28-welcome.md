Welcome, all. I am Jason Shipman, a software engineer in the Tampa/St\. Petersburg area. You can find out more about me [here](/about.html).  I have been itching to create a blog for a while and now it has begun!

This blog will be rooted pretty heavily in statically-typed functional programming, specifically with [Haskell](https://www.haskell.org/) and possibly some [Purescript](http://www.purescript.org/), [Idris](https://www.idris-lang.org/), \<insert some other ML-influenced language here\>, etc.  That being said, I enjoy programming regardless of static or dynamic typing.  I intend to learn a dynamic functional language at some point, so expect some posts about that too.  Whether that winds up being [Clojure](https://clojure.org/), [Erlang](https://www.erlang.org/), or [Elixir](https://elixir-lang.org/) remains to be seen.  More common languages like C++ and Javascript will likely crop up on this blog now and again as well.

A two-line Haskell program is worth a thousand words:

```haskell
main :: IO ()
main = putStrLn "Welcome!"
```
